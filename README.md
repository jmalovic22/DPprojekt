# DPprojekt
Implementacija AI agenta za taktičku igru koristeći Prolog i Python

Za projekt je potrebno imati instalirano sljedeće:
1. pyswip ( pip install pyswip )
2. python3 ( sudo apt install python3
3. pygame ( pip install pygame )

Za pokretanje terminal treba pozicionirati u skinuti folder  ( /DPprojekt ) i upisati naredba
  python3 ./main.py
